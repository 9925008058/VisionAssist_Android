# Security Policy

## Supported Versions

VisionAssist is currently in active. Security updates and fixes are provided for the latest version only.

| Version | Supported |
|----------|-----------|
| v1.0 | ✅ |
| Earlier versions | ❌ |

## Reporting a Vulnerability

The security and privacy of VisionAssist users are important.

If you discover a security vulnerability, please report it responsibly by opening a GitHub Issue with the label **security** or by contacting the repository maintainer through GitHub.

When reporting a vulnerability, please include:

- A clear description of the issue
- Steps to reproduce the vulnerability
- Potential impact
- Screenshots or logs (if applicable)

Please do not publicly disclose security vulnerabilities until they have been reviewed.

## Scope

This project focuses on:

- Object detection using Google Mlkit
- OCR-based text recognition
- Voice feedback and accessibility features
- Android application functionality

Reports related to third-party services, libraries, or platforms should be directed to their respective maintainers.

## Response Process

Security reports will be reviewed as soon as possible.

The typical process is:

1. Acknowledge the report
2. Verify the issue
3. Develop and test a fix
4. Release an update if necessary
5. Document the resolution

## Disclaimer

VisionAssist is an educational and open-source project. While reasonable efforts are made to improve security and reliability, no guarantees are provided regarding the absence of vulnerabilities.
